1. 테이블 생성

        표를 만들 때는 table 태그를 사용한다.
        thead : 테이블에서 제목 컬럼을 작성하는 구역
        tbody : 테이블에서 컬럼값을 작성하는 구역
        tfoot : footer contents를 작성하는 구역

        tr : 표의 행

thead, tbody, tfoot

    thead : 테이블에서 제목 컬럼을 작성하는 구역
    tbody : 테이블에서 컬럼값을 작성하는 구역
    tfoot : footer contents를 작성하는 구역

    sematic tag : 의미론적 태그는
    tag안의 내용이 어떤 의미인지를 나타내기 위해 사용하는 태그
    화면의 디자인에는 영향을 주지 않음.

    sematic tag를 사용해주면, 검색엔진의 봇이 페이지를 수집할 때
    보다 내용을 정확하게 수집해 가기 때문에 검색 사이트의 상단에
    페이지가 노출될 확률이 올라간다.
